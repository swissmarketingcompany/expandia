"How do I create a user with just email and password?"

To create a user with email and password, make a POST request to /api/v1/users with the required parameters: 

curl -X POST https://api.example.com/v1/users \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "mypassword123"
  }'

Note: Password must be at least 8 characters. The 'name' parameter is optional.
